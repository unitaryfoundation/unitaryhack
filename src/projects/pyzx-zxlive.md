PyZX (pronounce as Pisics) is a Python tool implementing the theory of ZX-calculus for the creation, visualisation, and automated rewriting of large-scale quantum circuits. ZXLive is a graphical user interface on top of PyZX to draw graphs or load circuits and apply ZX rules. Both are intended for experimenting, building proofs, helping to write papers, showing off, or simply learning about ZX and quantum computing.

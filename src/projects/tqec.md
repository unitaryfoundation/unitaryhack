TQEC is a Python-based open-source compiler that takes a logical-level quantum computation model represented as connected 3D primitive blocks and translates it into a detailed, fault-tolerant, physical-level circuit. The result is a Stim circuit with all the detailed information needed for simulation or to run on real quantum hardware. This enables both quantum algorithm designers and experimentalists to rapidly iterate and obtain exact low-level circuits, facilitating efficient performance simulation or experimental demonstration.

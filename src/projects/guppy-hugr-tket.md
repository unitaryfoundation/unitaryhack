# [Guppy](https://github.com/quantinuum/guppylang)
Guppy is a quantum programming language that is fully embedded into Python. It allows you to write high-level hybrid quantum programs with classical control flow and mid-circuit measurements using Pythonic syntax.

# [HUGR](https://github.com/quantinuum/hugr)
The Hierarchical Unified Graph Representation (HUGR, pronounced hugger) is a common representation for hybrid quantum circuits and operations, used at the backbone of 🐠 Guppy and 🎟️ TKET.

The project is focused on providing a fast and extensible framework for optimizing the next generation of quantum programs, that mix quantum circuits with classical control flow in the same package.

# [TKET](https://github.com/quantinuum/tket)
TKET is Quantinuum's compiler for modern structured programs that mix quantum and classical.
